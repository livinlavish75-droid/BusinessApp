LavishAI Tree Manager - Phone App / PWA Version

This version works on iPhone, Android, tablets, and desktop browsers.

HOW TO TEST LOCALLY:
1. Open index.html in Chrome or Edge.
2. For phone install features, the app needs to be hosted online or served through localhost.

HOW TO PUT IT ON YOUR PHONE:
1. Upload this folder to a website host, Netlify, Vercel, GitHub Pages, or your own domain.
2. Open the website link on your phone.
3. iPhone Safari: tap Share > Add to Home Screen.
4. Android Chrome: tap menu > Add to Home screen / Install app.

IMPORTANT:
- This version still saves data locally on each device.
- To sync data between Windows and phones, the next step is adding Firebase or Supabase.
